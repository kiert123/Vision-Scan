# 👁 VisionScan

## 📌 Project Overview
VisionScan is a web-based OCR system that allows users to upload images or PDFs, extract text, edit content, and save documents.

This project supports SDG Goal:
➡️ SDG 4 – Quality Education  
➡️ Helps digitize and edit printed learning materials.

---

## 🚀 Features
- Upload Image/PDF
- OCR Text Extraction (Tesseract)
- Convert PDF to editable text
- Edit extracted content
- Save documents
- View & Download files

---

## 🛠 Tech Stack
- PHP (Core)
- MySQL (Database)
- HTML/CSS (Frontend)
- Tesseract OCR
- XPDF (pdftotext)
- Composer (FPDF/FPDI)

---


## ⚙️ Installation Guide

1. Install XAMPP
2. Start Apache & MySQL
3. Move to htdocs
4. Install dependencies
5. Import database:
- Open phpMyAdmin
- Import `database.sql`

6. Configure:
- Edit `connection.php`

7. Install:
- Tesseract OCR
- XPDF (pdftotext)
