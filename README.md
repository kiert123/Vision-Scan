# 👁 VisionScan

## 📌 Project Overview
VisionScan is a web-based OCR system that allows users to upload images or PDFs, extract text, edit content, and save documents.

This project supports SDG Goal:
➡️ SDG 4 – Quality Education  
➡️ Helps digitize and edit printed learning materials.

---

## 🚀 Features
- Upload Image/PDF
- OCR Text Extraction (Tesseract)
- Convert PDF to editable text
- Edit extracted content
- Save documents
- View & Download files

---

## 🛠 Tech Stack
- PHP (Core)
- MySQL (Database)
- HTML/CSS (Frontend)
- Tesseract OCR
- XPDF (pdftotext)
- Composer (FPDF/FPDI)

---
